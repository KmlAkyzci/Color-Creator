# 🎨 Color Creator (Dinamik Renk Oluşturucu)

JavaScript ve modern CSS prensipleri kullanılarak geliştirilmiş, web geliştiriciler ve tasarımcılar için anlık olarak hex/rgb renk kodları üretmeyi, paletler oluşturmayı ve tek tıkla kopyalamayı sağlayan kullanıcı dostu bir web uygulaması.

## 🚀 Canlı Demo
Uygulamayı tarayıcınızda anlık olarak deneyimleyin:  
👉 https://kmlakyaz-color-generator.netlify.app/

---

## ✨ Özellikler
- **Rastgele Renk Üretimi:** Tek tıkla veya boşluk (space) tuşuyla tamamen rastgele ve birbiriyle uyumlu renk kodları üretebilme
- **Renk Paleti Oluşturma:** Tasarımlarınızda kullanabileceğiniz dinamik palet geçişleri.
- **Responsive Tasarım:** Mobil, tablet ve masaüstü cihazlarla tam uyumlu, modern ve minimalist arayüz.
